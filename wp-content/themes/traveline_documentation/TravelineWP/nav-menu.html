Creating & Setting a Menu

  1. Go to Appearance > Menus
  2. Click on Create a new menu
  3. Add some menu items to the menu
  4. Click on the Theme Locations tab select which menu you want to display in each area
  5. There is a more in depth guide for creating menus on the official WordPress website here
