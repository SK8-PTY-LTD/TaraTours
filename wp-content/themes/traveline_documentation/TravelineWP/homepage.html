Home Page Setup

  1. Go to Pages > Add New
  2. Add your Page Title in the Title field
  3. Select Default Template from the Template selection on the right side of the page
  4. Select Some Shortcode from FromShortcode for Homepage
  5. Edit Necessary text From your choosen shortcode
  6. If you want slider in your page than select yes form custom field Page Slider Option
  7. Click Publish
